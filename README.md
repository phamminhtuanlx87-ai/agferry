# 🚢 AGFerry Dashboard

Fullstack web application for internal management system.

## 🛠 Tech Stack

### Frontend

* React + TypeScript
* Vite
* TailwindCSS

### Backend

* ASP.NET Core Web API
* Entity Framework Core
* SQL Server

## 🚀 Features

* User registration
* Role-based system
* Department management
* Admin user creation
* Dashboard access control

## 💻 Run locally

### Backend

```bash
cd backend/agferry.api
dotnet run
```

API: http://localhost:5225

---

### Frontend

```bash
cd fontend/agferry-web
npm install
npm run dev
```

App: http://localhost:5173
